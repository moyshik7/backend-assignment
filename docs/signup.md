# Create a new user
  
Send a POST request to `/signup` with these credentials 
  
### Headers:
```form
content-type: application/json
```
  
  
### JSON values

```json
{
  "user" : {
    "username": "akio",
    "email": "moyshik7@gmail.com",
    "name": "Your Name",
    "password": "YourPassword"
  }
}
```
